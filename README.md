# 🩸 Blood Bank App

A modern and user-friendly Blood Bank application designed to connect blood donors with recipients in real-time. This platform simplifies the process of requesting and donating blood, helping save lives through quick and efficient communication.

---

## 🚀 Features

* 🧑‍🤝‍🧑 User Registration & Authentication
* 🩸 Create Blood Requests بسهولة
* ❤️ Become a Blood Donor
* 🔍 Search for Available Donors
* 📱 Responsive & User-Friendly Interface
* 🔔 Real-time Updates (optional if added)

---

## 🎯 Purpose

The main goal of this application is to digitize the traditional blood donation system and make it faster, more reliable, and easily accessible. It helps people in emergency situations quickly find donors and request blood without delays.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/blood-bank-app.git
```

Go to the project directory:

```bash
cd blood-bank-app
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add:

```env
MONGO_URI=your_database_connection
PORT=8000
JWT_SECRET=your_secret_key
```
